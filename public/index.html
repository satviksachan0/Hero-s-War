Chess-like Game

Note: To play the game, ensure that you open two browser tabs or windows. Each tab will represent a different player in the game. Both tabs need to be connected to the game server simultaneously for the game to function correctly.

Game Rules

Objective: Eliminate all of your opponent's pieces.

Setup: The game is played on a 5x5 grid. Each player controls 5 characters: Pawns (P), Hero1s (H1), and Hero2s (H2). Pieces are placed on the first and last rows of the board for each player.

Character Movements:

  • Pawn (P): Moves one block in any direction (Left, Right, Forward, or Backward).
  • Hero1 (H1): Moves two blocks straight in any direction. Can capture any opponent's piece in its path.
  • Hero2 (H2): Moves two blocks diagonally in any direction. Can capture any opponent's piece in its path.

Turn-Based Play: Players alternate turns. A move is only valid if it adheres to the character's movement rules and targets an empty cell or an opponent's piece.

Winning: The game ends when a player has eliminated all of the opponent's pieces. The player who achieves this first is declared the winner.
